# 账号基础设置

## 基础信息（SEO维度）

| 项目 | 要点 |
|------|------|
| **抖音名** | 区域+类目/行业（本身就是关键词搜索，SEO）。例：泗阳XX美食 |
| **抖音号** | 默认字母+数字组合，建议改为微信号或品牌相关ID（易限流，不要站外引流） |
| **头像** | 个人头像，真人出镜信任感更强 |
| **简介** | 用 DeepSeek 撰写，公式：❶我是谁 + ❷从事行业 + ❸带来价值，目的为变现 |
| **背景图** | 配合个人定位设计 |
| **作品四统一** | 封面、人物、文案、内容风格统一（不乱发，真实案例，贴近生活） |

> **"子母数字"**：即字母+数字组合。默认抖音号是系统分配的随机组合（如 `ABC123456`），建议修改。

## SEO 关键词布局

- 抖音名 = 核心关键词（区域+行业），直接影响搜索排名
- 简介中自然嵌入行业关键词
- 作品标题：关键词前置 + 价值明确 + 吸引点击
- 字幕中核心关键词自然出现 2-3 次
