# 电商选品与排品

## 橱窗带货开通

1. 开抖店获取自己的商品链接
2. 开直播 → 点击商品 → 右上角链接 → 粘贴自己的商品链接

### 企业号
- **开通**：创作者中心 → 全部 → 进阶服务 → 企业号（每年600元）
- **关闭**：企业服务中心 → 关闭企业号
- **相关工具**：抖店、抖音来客、商家主页

### 小店随心推
- 自己店的商品用小店随心推推广
- 付费与不付费流量的区别：付费是锦上添花，不能依赖

## 标准排品结构

| 链接类型 | 占比 | 利润率 | 过款时间 |
|---------|------|--------|---------|
| 引流品/福利品 | 1号链接 | 微赚/微亏 | 开播15分钟内，数据低迷时返场 |
| 爆品 | 2号链接 | 10-30% | 开播15分钟后，持续返场 |
| 利润款 | 3号链接 | 30-70% | 穿插在引流品和爆品之间 |
| 测试品 | 7-10号链接 | — | 重点测试，数据好再推爆 |

## 排品节奏口诀

```
引流品 → 承接极速流，稳住在线
爆品 → 拉升GPM，矫正流量标签
利润品 → 实现盈利
```

**相邻品之间要有关联**（如T恤→裤子→连衣裙），不竞争。

## 直播转化产品节奏

- 对比：平时价格 vs 现在价格
- 上链接时可用小号进直播间制造热度
- 逼单技巧：制造稀缺性（库存/时间/数量限制）
