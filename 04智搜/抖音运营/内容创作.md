# 短视频内容创作

## 封面/开头/文案公式

| 公式 | 作用 | 示例 |
|------|------|------|
| 疑问 + 求真 | 吸引认知 | "你知道XX其实是这样做的吗？" |
| 数字 + 对比/方式 | 激发好奇心 | "3个方法，让XX从A变成B" |
| 悬疑 + ... | 制造悬念 | "千万别这样做，否则..." |
| 情绪 + 情景 | 带入感 | "如果你也在经历XX，一定要看完" |

> "数字+对比/方式=异化好奇心"——笔记中的"异化"应为"激发"的笔误或口语化表达。

## 拍摄技巧

### 基本设置
- 调整分辨率和帧率
- 打开水平仪/网格线辅助构图
- 高清发布开关打开

### 景别与镜头
- 景别：近景、远景、全景、中景、特写
- 开头和结束用远全景，中间自由切换
- 分镜头拍摄，丰富画面节奏

### 稳定性
- 手持拍摄：用身体作为支点
- 多用固定镜头减少晃动

### 构图方法

| 构图 | 效果 |
|------|------|
| **三角形构图** | 画面中三点构成三角形，正三角=稳定，斜三角=灵活，倒三角=张力 |
| **建筑构图** | 利用建筑线条、对称、门框等几何结构辅助构图 |
| **低角度构图** | 仰拍增强视觉冲击力 |
| **三分法构图** | 单个人物不放正中间 |

> **建筑构图说明**：利用建筑本身的几何形态（对称、线条、框架、重复排列）来构图。例如用门框做"框式构图"、用建筑线条做"引导线构图"、用对称结构做"对称构图"。拍摄短视频时，建筑环境是天然的构图辅助工具。

## 文案提升技巧

- **关键词搜索流量**：首页→猜你想搜→相关搜索（蓝色词为热词）
- **创作灵感凑热度**：用创作灵感功能看平台推荐热门选题
- **评论区加热**：评论里加热词，后面系统会自动生成"小蓝词"（搜索联想词）

## 内容标签优化

- 标题文案：吸引人、关注率高的词语
- 内容信息：按385法则安排节奏
- 关键词+话题+@相关领域达人
- 话题不要自己创建，用热度高的现有关联话题

## 流量转化路径

```
公域流量 → 私域流量 → 个人/线下
矩阵剪辑后同时发，两三天全覆盖
作品天天发 → 积累粉丝 → 转化到线下
```
