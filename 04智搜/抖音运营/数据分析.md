# 数据分析与账号诊断

## "蓝线包裹红线" 解读

> 来自抖音创作者后台 → 数据中心 → 五星雷达图（账号诊断）。

- **蓝色线（内部/基准）** = 同类创作者的平均数据水平（同赛道账号均值）
- **红色线（你的数据）** = 你自己的实际表现

### 诊断结论

| 状态 | 含义 | 对策 |
|------|------|------|
| 🔴 红线被蓝线包裹 | 数据低于同类平均水平 → 低质账号风险 | 针对性优化弱项 |
| 🟢 红线跳出蓝线 | 数据超越同类平均 → 有机会突破流量层级 | 加大优质内容投入 |
| ⭐ 五指标饱满 | 账号权重高，基础播放量大 | 持续产出 |

### 判断标准

查看"超越同类创作者百分比"：
- ✅ ≥60% → 及格/优质
- ❌ <60% → 未达标，需优化

## 核心指标与优化方向

| 指标 | 若偏低 | 优化方法 |
|------|--------|---------|
| 播放量 | 作品质量或违规问题 | 提升内容质量，避免违规 |
| 完播率 | 开头不够吸引 | 缩短内容长度，多切换镜头 |
| 粉丝增长 | 未引导关注 | 视频中明确引导关注、三连 |
| 互动率 | 缺乏话题性 | 增加讨论性话题、争议性提问 |
| 投稿量 | 更新频率不足 | 保持稳定更新频率 |

## 直播间数据诊断

- **主播诊断**：记录十场直播数据，好的坏的都要记，找问题
- **罗盘达人**：看直播数据，**停留一分钟才合格**

## 2026年数据权重变化

| 旧权重 | 新权重 |
|--------|--------|
| 点赞为主 | **收藏率 > 复访率 > 铁粉完播** |
| 24小时爆发 | **7天长效赛马** |
| 标签推流 | 行为预测推流 |
| 可蹭热点 | 去泛流量，重垂直精准 |

## 搜索流量监控

成熟账号搜索流量占比应达 **30% 以上**（低于 15% 说明 SEO 不足）。

需要关注：
- **关键词转化率**：流量高但转化率低 → 调整转化引导方式
- **搜索用户停留时长**：应高于推荐用户，否则需优化内容价值密度
