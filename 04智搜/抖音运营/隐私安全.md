# 账号隐私与安全设置

## "推荐可能认识的人 — 一开二关" 解读

> **常见误解**："一开二关"不是"开一项关两项"，而是指：
>
> 1. **（一）开** → 进入隐私设置页面
> 2. **（二）关①** → 关闭「把我推荐给可能认识的人」
> 3. **（三）关②** → 关闭通讯录权限（系统层面）

## 完整隐私设置清单

| 设置项 | 操作 | 路径 |
|-------|------|------|
| 私密账号 | 关闭 | 隐私设置 |
| 黑名单 | 清空 | 隐私设置 → 黑名单 |
| 关注/粉丝列表 | 设为私密 | 隐私设置 |
| 把我推荐给可能认识的人 | 🔴 关闭 | 隐私设置 → 找到我的方式 |
| 在他人关注/粉丝列表公开出现 | 🔴 关闭 | 隐私设置 → 找到我的方式 |
| 同城展示 | 🔴 关闭（可选） | 隐私设置 |
| 通讯录权限 | 🔴 关闭 | 系统权限 → 通讯录权限 |

## 为什么要拉黑熟人

> 这不是社交洁癖，而是基于算法的务实操作。

**核心逻辑**：保护冷启动数据纯净。

熟人不是目标受众，他们点开视频：
- **完播率极低**（看两眼划走）→ 算法收到"内容不吸引人"信号
- **评论不精准**（"哈哈你在干嘛"）→ 无法帮算法识别内容标签
- **社交关系权重干扰** → 视频优先推给好友圈，消耗初始推荐量却无法破圈

**结果**：视频在社交圈层内就"死"了，触达不到真正潜在粉丝。很多成功创作者拉黑 100-200 个熟人后做起来的。

> **例外**：如果你的内容就是面向本地生活（探店、同城服务），熟人推荐反而是助力。

## 安全维护

- **自助清粉**：账号安全 → 安全中心 → 更多功能 → 自助清粉（**每周清理一次**）
- **限流查询**：抖音灵机 → 查被谁举报、是否被限流
- **违规检测**：发文助手（免费检测违规）

## 直播屏蔽词设置

路径：直播中心 → 直播设置 → 屏蔽词管理

> **关于"84"**：笔记中的"84"并非指84个屏蔽词或特定敏感数字，可能是随口举例或记录误差。实际上抖音直播屏蔽词**上限10个**。

建议屏蔽词：骗人、忽悠、假的、太贵、质量差、别买了、退货
