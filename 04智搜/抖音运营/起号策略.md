# 起号策略

## 赛马机制

同时发布多个类目的作品（如男装/女装/童装/老年装各一个），看哪条数据好。通过数据中心→粉丝画像分析集中年龄段，确定从事行业。

**核心**：先测赛道，再聚焦。

## 三天测试期

起号保持更新，3天测试期，看哪个赛道数据好用哪个。

| 赛道 | 说明 |
|------|------|
| 🌾 **三农**（农民/农业/农村） | 平台扶持赛道 |

## dou+ 使用原则

- **锦上添花**：只有自然流量数据本身已不错时再投
- **不能长期投**：经常投 dou+ 会导致没有自然流量
- 主要用途：数据好的作品加热放大，不是用来拯救烂作品

## 发布时间策略

### 常规黄金时段
- 早 6:00-7:30（通勤早高峰）
- 中 10:30-11:00（午餐决策期）
- 晚 20:00-21:00（黄金时段，竞争最激烈）

### "早餐晚上发，午餐早上发" 反逻辑策略

> 这不是笔误，是**错峰发布+提前种草**策略：

1. **提前种草**：晚上发早餐 → 睡前收藏 → 次日早上看（收藏率高）
2. **错峰竞争**：早上大家都在发早餐，晚上没人发 → 独占曝光窗口
3. **审核缓冲**：发出去需 2-3 小时审核跑量 → 晚上发，早上进推荐流
4. **反差吸睛**：深夜看到诱人早餐 → 反差强 → 互动意愿高

> ⚠️ 新手建议先从常规时间起步，跑通数据后再测试错峰策略。

## 核心原则

- **同手机 ≤ 3 个账号**（超过限流）
- **搬运必须二创**：大幅修改后再发
- **发布频率**：一天最低一条。抖音推流有固定分配值，集中发一个作品流量更集中
- **搜索流量 = 精准流量**（用户主动搜索意味着已有明确需求）
- **话题策略**：一个作品加 5 个热门话题（不能自己创建），从热点宝/巨量算数/创意灵感找

## 作品分类

| 类型 | 特点 |
|------|------|
| 电影剪辑 | 需注意版权 |
| 搬运剪辑 | 必须大幅二创 |
| 对口型 | 门槛低，但难持久 |
| 知识分享 | 垂直度高，粉丝粘性强 |
| 搞笑类 | 受众广，但变现路径长 |
| **美食分享** | **推荐起号**，农村美食效果更佳 |

## 数据验证

- 7天定位期，确定目标领域和身份
- 生活类易做，题材多
- 对农村三农有平台扶持
- 三天喜好标签测试：同类型账号互动（打字、送礼物），固定时间发布
- 四天完善主页
- 五天涨粉丝
- 六天发作品（4个垂直作品）+ 开直播加热
