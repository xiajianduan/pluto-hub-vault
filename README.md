# Pluto Hub Vault

一个基于 Obsidian 的个人知识库，集成了 **Pluto Hub** 插件（v2.0.0），用于管理笔记、国学经典、工作文档和代码模块。

## 仓库结构

```
.
├── 01草稿/          # 草稿与进行中的笔记
├── 02国学/          # 国学经典（道德经、孙子兵法、词谱详解）
├── 03工作/          # 工作笔记
├── 04智搜/          # 智能搜索内容（抖音运营知识体系）
├── 98图片/          # 图片资源
├── 99系统/          # 系统配置（皮肤、示例、插件文档截图）
├── .obsidian/       # Obsidian 配置及 Pluto Hub 插件
├── CLAUDE.md        # Claude Code 项目指南
├── README.md        # 本文件
└── README_en.md     # English README
```

## Pluto Hub 插件

内置 [Pluto Hub](.obsidian/plugins/pluto-hub/) 插件（v2.0.0）提供本地代码模块的创建、编辑与运行功能，支持主题定制和代码侧边栏管理。

- **插件 ID**: `pluto-hub`
- **最低版本**: Obsidian 1.7.2
- **平台**: 桌面端 + 移动端
- **构建方式**: 已预编译为 `main.js`，无需额外构建步骤

## 内容概览

| 目录 | 内容 | 说明 |
|------|------|------|
| `01草稿/` | 培训笔记 | 非正式、进行中的笔记 |
| `02国学/` | 道德经、孙子兵法、词谱详解 | 中国传统经典研读 |
| `03工作/` | Charlotte's Web 等 | 英文工作笔记 |
| `04智搜/` | 抖音运营知识体系（10 篇） | 内容创作、算法、运营策略等 |
| `99系统/` | 皮肤、示例、说明文档 | Obsidian 主题与插件文档素材 |

## 快速开始

1. 用 Obsidian（v1.7.2+）打开此仓库作为 vault
2. Pluto Hub 插件将自动加载
3. 如遇插件安全性提示，请信任作者以启用
