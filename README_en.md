# Pluto Hub Vault

An Obsidian-based personal knowledge base, integrated with the **Pluto Hub** plugin (v2.0.0) for managing notes, Chinese classics, work documents, and code modules.

## Repository Layout

```
.
├── 01草稿/          # Drafts & in-progress notes
├── 02国学/          # Chinese classics (Tao Te Ching, Art of War, Ci Poetry)
├── 03工作/          # Work notes
├── 04智搜/          # Smart search content (Douyin operations knowledge base)
├── 98图片/          # Image assets
├── 99系统/          # System config (skins, samples, plugin docs)
├── .obsidian/       # Obsidian config & Pluto Hub plugin
├── CLAUDE.md        # Claude Code project guidelines
├── README.md        # Chinese README
└── README_en.md     # This file
```

## Pluto Hub Plugin

The built-in [Pluto Hub](.obsidian/plugins/pluto-hub/) plugin (v2.0.0) enables local code module creation, editing, and execution, along with theme customization and a code sidebar manager.

- **Plugin ID**: `pluto-hub`
- **Min Version**: Obsidian 1.7.2
- **Platform**: Desktop + Mobile
- **Build**: Pre-compiled `main.js`, no build step required
